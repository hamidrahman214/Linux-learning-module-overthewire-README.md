## Level 10 - 11
***Aim:*** Find the password from file data.txt, which contains base64 encoded data.

***Action taken:*** 
- used the `ls` command to list the contents
- Seen that the `data.txt` file was in the current working directory
- used the `base64 -d` command to decode the file and it printed the password

```bash
base64 -d data.txt
```
***Level 11 Password:*** `dtR173fZKb0RRsDFSGsg2RWnpNVj3qRr`
